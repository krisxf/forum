# 项目背景

# 所需环境

- redis
- rabbitmq
- minio
- mysql
- mail

# 实现功能

## 安全效验
- 注册时使用第三方的验证码效验
- 自定义注解实现的接口限流
- jwt进行用户的认证
- RBAC 权限管理设计

## 用户功能

- 登入登出
- 查询，评论文章
- 查看已评论，已收藏，已发布的信息
- 点赞，取消赞

## 管理员功能

- 用户的管理
- 首页轮播图的管理
- 文章管理
- 分类管理
- 评论管理
- 标签管理(待做)

## 其他

- 使用消息队列RabbitMQ进行异步通知(包括邮件发送验证码，文章被评论通知)
- 使用Minio进行图片的存储
- Redis 实现排行榜
- 敏感词汇过滤
